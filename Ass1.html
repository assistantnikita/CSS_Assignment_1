What is CSS?

  1. CSS Stands for Cascading Style Sheet.
  2. Css describe how html, elements are to be displayed on screen ,paper or in other media.
  3. css is not necessaryCSS saves a lot of work. It can control the layout of multiple webpage all at once.
  4. External Stylesheet are stored in CSS files.

CSS can be added to HTML document in 3 ways:

  • Inline - by using the style attributes inside HTML Element.
  • Internal - by using a style element in the head section.
  • External - by using a link element two link to an external css file.
  • The most common wy to add CSS. is to keep the style in external CSS files. how ever, in this tutorial we will use inline and internal style, beacause this is easier to demonstrate and easier for you to try it yourself.

When comapring CSS class vs ID , the differnece is that CSS class applied a style to multiple elements. ID is also special in that you can use a special URL. to link directly to an element and it is used by javascript.

In CSS, selectors are used to target a specific element or range of elements on a webpages. Once an element has been targeted, a style or set of style can be applied to the elemets.

There is a wide range of selectors available.Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
